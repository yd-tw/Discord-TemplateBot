## 專案介紹

這個專案提供各種基礎的 Discord 機器人模板。由 CodeCat 程式貓社群支援維護。

## 技術細節

- Python
- Discord.py

## 製作歷程

- 初代設計 提供一個通用模板
- 重構設計 提供各種模板供選擇
- 目前版本 `1.5.0`

## 使用方法

1. `clone` 這個儲存庫
2. 將 Bot Token 添加至 `.env.example` 中並重新命成 `.env`
3. 安裝 `requierments.txt` 中的依賴項並執行 `main.py` (Windows 使用者可以執行 `run.bat` 自動啟動)

若要退出 Python venv 環境可以執行 `deactivate`。
